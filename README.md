# Study Planner

This project is a web-based homework planner that allows students to track their assignments and monitor their progressin an efficient and organized manner.

Users are able to:
- Register and log in
- Add assignments
- Track assignment progress (not started, in-progress, completed)
- Organize assignments by class

The database model consists of three entities:
Student, Class, and Assignment.
